## Intro

The Vendor Directory provides a comprehensive list of vendors, service providers, and partners used by the Everyday Homebuyers workstream. This includes contractors, title companies, attorneys, inspectors, and other service providers. Maintaining an up-to-date vendor directory enables efficient vendor management, quality service delivery, and strong partnerships.

## Purpose

Centralize vendor information, contact details, and service descriptions to enable efficient vendor management and service coordination. This ensures team members can quickly find and contact appropriate vendors.

## When This Applies

- Finding vendors for specific services
- Contacting vendors for quotes or work
- Managing vendor relationships
- Evaluating vendor performance
- Adding new vendors

## KPIs or SLAs

- **Directory Completeness**: 100% of active vendors listed
- **Directory Accuracy**: 95%+ accurate information
- **Vendor Performance**: High performance ratings
- **Vendor Satisfaction**: Positive vendor relationships

## Example Scenarios

**Scenario 1: Finding Vendor**
Team needs contractor for property rehab. Checks vendor directory, finds general contractor in area. Contacts vendor, gets quote, schedules work. Vendor directory enables quick vendor finding.

**Scenario 2: Vendor Performance**
Vendor performance declining. Team provides feedback, tracks issues. Operations reviews, works with vendor to improve. If no improvement, replaces vendor. Vendor directory updated. Quality maintained.

**Scenario 3: New Vendor**
New vendor identified, good reputation, competitive pricing. Evaluated, added to directory. Team uses vendor, provides feedback. Vendor performs well, becomes preferred vendor. Directory updated.
