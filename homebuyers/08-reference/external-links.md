## Intro

External Links provides a centralized directory of external resources, tools, websites, and platforms used by the Everyday Homebuyers workstream. This includes industry resources, training materials, tools, and reference sites. Maintaining this directory enables quick access to external resources and ensures team members know where to find needed information.

## Purpose

Centralize external links and resources for easy access and reference. This ensures team members can quickly find and access external tools, resources, and information.

## When This Applies

- Finding external resources
- Accessing tools and platforms
- Research and learning
- Reference during work
- Sharing resources with team

- Check links regularly

## KPIs or SLAs

- **Link Accuracy**: 95%+ of links working
- **Directory Completeness**: All key resources included
- **Link Currency**: Directory updated regularly
- **Link Usage**: High usage of directory

## Example Scenarios

**Scenario 1: Finding Resource**
Team member needs MLS access. Checks external links directory, finds MLS link and access instructions. Accesses MLS, finds needed information. Directory enables quick resource access.

**Scenario 2: Broken Link**
Team member finds broken link. Reports to Operations. Operations checks link, updates or removes. Directory updated. Link accuracy maintained.

**Scenario 3: New Resource**
New training resource identified. Operations evaluates, adds to directory with description and access info. Shares with team. Directory updated, team benefits from new resource.
