## Intro

The Call Script Template provides a standardized format for creating call scripts used in lead management, acquisitions, and dispositions. This template ensures consistent, effective communication while allowing for personalization and adaptation. Using this template enables script creation that balances structure with flexibility.

## Purpose

Provide a reusable template for creating call scripts that ensures consistent structure, effective communication, and proper handling of common scenarios. This enables efficient script creation and consistent execution.

## When This Applies

- Creating new call scripts
- Updating existing scripts
- Training on script usage
- Standardizing communication
- Improving call effectiveness

## Process / Framework / Workflow

### Call Script Template Structure

**Script Header:**
- Script Name
- Purpose
- When to Use
- Target Audience
- Estimated Duration

**Script Sections:**
1. Opening
2. Introduction
3. Value Proposition
4. Qualification/Discovery
5. Objection Handling
6. Closing
7. Follow-Up

### Script Components

**Opening:**
- Greeting
- Name confirmation
- Company introduction
- Reason for calling
- Permission to proceed

**Introduction:**
- Who you are
- Why you're calling
- What you do
- How you can help

**Value Proposition:**
- Key benefits
- What makes you different
- Why they should care
- How you solve their problem

**Qualification/Discovery:**
- Key questions
- Information gathering
- Needs assessment
- Fit evaluation

**Objection Handling:**
- Common objections
- Response strategies
- Handling techniques
- When to persist vs. back off

**Closing:**
- Next steps
- Commitment
- Follow-up plan
- Thank you

**Follow-Up:**
- What happens next
- Timeline
- Contact information
- Additional resources

### Voice/Tone Examples

**Warm and Friendly:**
- "Hi [Name], I hope you're having a great day!"
- Use positive language
- Show genuine interest
- Maintain upbeat tone

**Professional and Confident:**
- "I'm calling from [Company], and I believe we can help you."
- Use clear, direct language
- Show expertise
- Maintain professional tone

**Empathetic and Understanding:**
- "I can only imagine how stressful this must be."
- Acknowledge their situation
- Show compassion
- Maintain supportive tone

### Close Techniques

**Technique 1: Assumptive Close**
- Assume positive outcome
- Use confident language
- Guide toward decision
- Example: "When would be a good time to schedule a showing?"

**Technique 2: Alternative Close**
- Offer choices
- Both choices lead to next step
- Example: "Would you prefer to see the property this week or next?"

**Technique 3: Summary Close**
- Summarize value and benefits
- Reinforce key points
- Ask for commitment
- Example: "So we've discussed [benefits]. Would you like to move forward?"

## Tools Used

- **Script Template**: This template
- **PhoneBurner**: Script display during calls

- **Script Library**: Centralized script storage

## Role Responsibilities

**Script Creators**
- Create scripts using template
- Ensure completeness and effectiveness
- Test scripts before deployment
- Update scripts based on feedback

**Callers**
- Use scripts as guide, not verbatim
- Personalize appropriately
- Adapt to conversation flow
- Provide feedback on scripts

**Managers**
- Review script effectiveness
- Provide coaching on usage
- Optimize scripts based on results
- Train team on script usage

## KPIs or SLAs

- **Script Usage**: High adherence to scripts
- **Call Effectiveness**: Improved conversion rates
- **Script Quality**: Effective and complete scripts
- **Script Updates**: Updated based on performance

## Example Scenarios

**Scenario 1: Creating New Script**
Need script for new scenario. Use template, fill in sections, include openings, value prop, objections. Test script, refine based on feedback. Deploy to team, monitor effectiveness.

**Scenario 2: Using Script Effectively**
Caller uses script as guide, personalizes for conversation, adapts to homeowner responses. Maintains script structure while being natural. Effective call, positive outcome.

**Scenario 3: Script Improvement**
Script not generating desired results. Review script, identify issues, update using template. Test updated script, deploy. Results improve.
