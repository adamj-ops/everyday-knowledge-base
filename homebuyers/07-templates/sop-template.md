## Intro

This SOP Template provides a standardized format for creating Standard Operating Procedures (SOPs) within the Everyday Homebuyers workstream. Using this template ensures consistency, completeness, and clarity across all SOPs. This template includes all required sections and best practices for effective process documentation.

## Purpose

Provide a reusable template for creating SOPs that ensures consistent structure, complete information, and effective process documentation. This enables efficient SOP creation and maintenance.

## When This Applies

- Creating new SOPs
- Updating existing SOPs
- Standardizing process documentation
- Training on SOP creation
- Ensuring SOP consistency

## Tools Used

- **SOP Template**: This template
- **Documentation System**: SOP storage and management
- **Process Mapping Tools**: Visual workflow creation
- **Version Control**: Track SOP changes

## Role Responsibilities

**Process Owners**
- Create SOPs using template
- Maintain SOP accuracy
- Update SOPs when processes change
- Ensure SOP completeness

**Team Members**
- Follow SOPs consistently
- Provide feedback on SOPs
- Suggest SOP improvements
- Report process issues

**Operations**
- Ensure SOP template usage
- Monitor SOP quality
- Optimize template as needed
- Support SOP creation

## KPIs or SLAs

- **SOP Completeness**: 100% include all required sections
- **SOP Accuracy**: 95%+ accurate and current
- **SOP Usage**: High adherence to SOPs
- **SOP Updates**: Updated when processes change

## Example Scenarios

**Scenario 1: Creating New SOP**
Process owner uses template to create new SOP. Fills in all sections, includes workflow steps, defines roles and SLAs. SOP reviewed, approved, published. Team uses SOP consistently.

**Scenario 2: Updating Existing SOP**
Process changes, SOP needs update. Process owner uses template structure, updates relevant sections, maintains consistency. Updated SOP reviewed, approved, published. Team notified of changes.

**Scenario 3: SOP Quality Review**
Operations reviews SOPs for quality. Checks template adherence, completeness, accuracy. Identifies improvements, works with process owners to enhance SOPs. SOP quality improved.
