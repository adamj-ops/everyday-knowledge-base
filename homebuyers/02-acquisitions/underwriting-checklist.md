## Intro

The Underwriting Checklist ensures thorough, consistent evaluation of every property opportunity. This checklist guides acquisitions specialists through all required steps, data points, and analyses needed to make informed acquisition decisions. Following this checklist minimizes errors, ensures nothing is missed, and enables accurate offer generation.

## Purpose

Provide a comprehensive checklist that ensures every property is underwritten thoroughly and consistently. This checklist serves as both a process guide and a quality control tool.

## When This Applies

- Underwriting any qualified property
- Training new acquisitions team members
- Ensuring consistency across team
- Auditing underwriting quality
- Standardizing evaluation process

## Process / Framework / Workflow

### Underwriting Checklist

#### Section 1: Property Information & Verification

- [ ] Property address verified and accurate
- [ ] Property type confirmed (single family, multi-family, etc.)
- [ ] Square footage verified
- [ ] Bedrooms and bathrooms confirmed
- [ ] Lot size documented
- [ ] Year built verified
- [ ] Property tax information obtained
- [ ] HOA information (if applicable) gathered
- [ ] Property history reviewed (previous sales, ownership)

#### Section 2: Location & Neighborhood Analysis

- [ ] Neighborhood identified and assessed
- [ ] School district information gathered
- [ ] Crime statistics reviewed (if relevant)
- [ ] Nearby amenities documented
- [ ] Market trends in area analyzed
- [ ] Comparable neighborhood sales reviewed
- [ ] Future development plans checked (if relevant)
- [ ] Location desirability assessed

#### Section 3: Comparable Sales Analysis (Comps)

- [ ] Minimum 3 comparable properties identified
- [ ] Comps sold within last 6 months (preferred)
- [ ] Comps within 1 mile radius (preferred)
- [ ] Similar square footage (Â±20%)
- [ ] Similar property type and features
- [ ] Comps adjusted for differences:
  - [ ] Size differences
  - [ ] Condition differences
  - [ ] Feature differences (garage, updates, etc.)
  - [ ] Location differences
- [ ] Price per square foot calculated
- [ ] ARV range determined (conservative to optimistic)
- [ ] ARV methodology documented

**See Also:** [Comps Analysis Guide](./comps-analysis-guide.md), [ARV Calculation Guide](./arv-calculation-guide.md)

#### Section 4: Property Condition Assessment

- [ ] Property condition information reviewed
- [ ] Photos examined (if available)
- [ ] Inspection reports reviewed (if available)
- [ ] Property condition categories assessed:
  - [ ] Structural (foundation, roof, walls)
  - [ ] Systems (HVAC, plumbing, electrical)
  - [ ] Exterior (siding, windows, doors, landscaping)
  - [ ] Interior (floors, walls, kitchen, bathrooms)
  - [ ] Cosmetic (paint, fixtures, finishes)
- [ ] Major issues identified and documented
- [ ] Property condition rating assigned (1-10 scale)

#### Section 5: Repair Cost Estimation

- [ ] Repair costs estimated by category:
  - [ ] Structural repairs
  - [ ] Systems repairs/updates
  - [ ] Exterior work
  - [ ] Interior updates
  - [ ] Cosmetic improvements
  - [ ] Cleanup and debris removal
  - [ ] Landscaping
- [ ] Contractor quotes obtained for major items (if needed)
- [ ] Material costs estimated
- [ ] Labor costs estimated
- [ ] Permit costs included (if needed)
- [ ] Contingency added (10-15%)
- [ ] Total repair cost calculated
- [ ] Repair cost breakdown documented

#### Section 6: Financial Analysis

- [ ] ARV determined (from comps analysis)
- [ ] Repair costs totaled (from repair estimation)
- [ ] Holding costs estimated:
  - [ ] Property taxes
  - [ ] Insurance
  - [ ] Utilities
  - [ ] Maintenance
  - [ ] Timeline-based costs
- [ ] Closing costs estimated (purchase)
- [ ] Closing costs estimated (disposition)
- [ ] Desired profit margin applied
- [ ] MAO (Maximum Allowable Offer) calculated:
  - [ ] MAO = ARV - Repair Costs - Holding Costs - Desired Profit - Closing Costs
- [ ] Profit potential calculated
- [ ] ROI calculated
- [ ] Deal viability assessed

#### Section 7: Risk Assessment

- [ ] Market risk assessed (market conditions, trends)
- [ ] Property risk assessed (condition, location, issues)
- [ ] Timeline risk assessed (holding period, market changes)
- [ ] Competition risk assessed (other buyers, market activity)
- [ ] Seller risk assessed (motivation, reliability, situation)
- [ ] Overall risk level assigned (Low, Medium, High)
- [ ] Risk mitigation strategies identified (if needed)

#### Section 8: Deal Decision

- [ ] MAO is positive (profitable)
- [ ] Profit margin meets minimum thresholds
- [ ] Risks are acceptable
- [ ] Deal fits acquisition criteria
- [ ] Decision made: Proceed, Pass, or Need More Info
- [ ] Decision rationale documented
- [ ] Manager approval obtained (if required)

#### Section 9: Offer Development (if proceeding)

- [ ] Initial offer amount determined
- [ ] Offer positioning strategy developed
- [ ] Negotiation room planned
- [ ] Offer terms developed:
  - [ ] Price
  - [ ] Closing timeline
  - [ ] Contingencies
  - [ ] Special conditions
- [ ] Offer explanation prepared
- [ ] Manager approval obtained (if required)

#### Section 10: Documentation

- [ ] All underwriting data entered into CRM
- [ ] Comps analysis documented
- [ ] Repair estimate documented
- [ ] Financial analysis documented
- [ ] Risk assessment documented
- [ ] Decision and rationale documented
- [ ] Offer strategy documented (if proceeding)
- [ ] All supporting documents attached

### Quality Standards

**Completeness:**
- All checklist items completed
- No missing information
- All required data points gathered

**Accuracy:**
- Data verified and accurate
- Calculations double-checked
- Assumptions documented

**Thoroughness:**
- Sufficient comps analyzed
- Repair costs realistic
- Risk factors considered

**Documentation:**
- All work documented in CRM
- Rationale clearly explained
- Supporting documents attached

## Tools Used

- **Underwriting Checklist**: This document (digital or printed)
- **CRM System**: Data entry and documentation
- **Comps Analysis Tools**: MLS, public records, valuation software
- **Repair Estimators**: Cost databases, contractor networks
- **Deal Analyzer**: Comprehensive underwriting spreadsheet
- **Calculator Tools**: Financial analysis and MAO calculation

## Role Responsibilities

**Acquisitions Specialists**
- Complete checklist for every property
- Ensure all items checked off
- Document work thoroughly
- Seek help when needed
- Maintain quality standards

**Acquisitions Manager**
- Review checklist completion
- Audit underwriting quality
- Provide guidance and support
- Ensure standards maintained
- Optimize checklist based on results

## KPIs or SLAs

- **Checklist Completion**: 100% of items completed for every deal
- **Underwriting Accuracy**: 95%+ of deals meet profit projections
- **Time to Complete**: 24-48 hours for standard properties
- **Quality Score**: Regular audits show high quality standards

## Example Scenarios

**Scenario 1: Standard Underwriting**
Acquisitions specialist receives qualified lead. Completes checklist systematically: verifies property info, runs comps analysis, estimates repairs, calculates MAO. All items checked, documentation complete. Decision: Proceed with offer. Offer generated and presented.

**Scenario 2: Complex Property**
Property has unique features, limited comps. Specialist completes checklist but notes challenges. Gets manager support for comps analysis. Uses alternative valuation methods. Completes all checklist items with notes on methodology. Decision: Proceed with conservative offer.

**Scenario 3: Quick Pass**
Property clearly outside target market, unrealistic seller expectations. Specialist completes initial checklist items quickly, identifies deal-breaker early. Documents rationale, makes decision to pass. Saves time by not completing full analysis.
