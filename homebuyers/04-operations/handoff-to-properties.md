## Intro

The Handoff to Properties process ensures smooth transfer of acquired properties from Acquisitions to the Properties team for rehabilitation. This handoff includes all necessary information, documentation, and coordination to enable efficient rehab planning and execution. Effective handoffs prevent delays, ensure quality, and maintain momentum from acquisition through sale.

## Purpose

Establish a standardized handoff process that ensures Properties team receives complete, accurate information needed to plan and execute property rehabilitation efficiently. This enables seamless transitions and optimal rehab outcomes.

## When This Applies

- Property acquisition closed and needs rehab
- Transferring property from Acquisitions to Properties
- Ensuring complete property information transfer
- Coordinating rehab planning and execution

## Process / Framework / Workflow

### Handoff Trigger

**When Handoff Occurs:**
- Property closing completed
- Property requires rehabilitation before sale
- Acquisitions team confirms property ready for handoff
- **Timeline:** Within 24 hours of closing

### Handoff Checklist

**Property Information:**
- [ ] Property address (verified)
- [ ] Property type and details
- [ ] Square footage, bedrooms, bathrooms
- [ ] Lot size and characteristics
- [ ] Year built and construction details
- [ ] Current property condition

**Financial Information:**
- [ ] Acquisition price
- [ ] Estimated ARV
- [ ] Estimated repair costs (from underwriting)
- [ ] Target sale price
- [ ] Budget allocation for rehab
- [ ] Profit margin targets

**Property Condition:**
- [ ] Property condition report
- [ ] Inspection reports (if available)
- [ ] Photos (current condition)
- [ ] Known issues and concerns
- [ ] Repair priorities identified

**Rehab Scope:**
- [ ] Detailed repair list from underwriting
- [ ] Repair estimates by category
- [ ] Rehab priorities and sequencing
- [ ] Special considerations or requirements
- [ ] Timeline expectations

**Documentation:**
- [ ] Purchase agreement
- [ ] Title documents
- [ ] Inspection reports
- [ ] Property photos
- [ ] Underwriting analysis
- [ ] Any relevant notes or context

**Access & Logistics:**
- [ ] Property keys or access codes
- [ ] Property location and directions
- [ ] Utilities status (on/off, accounts)
- [ ] Property security status
- [ ] Any access restrictions or issues

### Handoff Process

**Step 1: Preparation (Acquisitions)**
1. Complete property closing
2. Gather all handoff information
3. Complete handoff checklist
4. Prepare handoff package
5. Schedule handoff meeting (if needed)

**Step 2: Handoff Execution**
1. Transfer property information to Properties team
2. Provide all documentation
3. Transfer property keys/access
4. Review property condition and scope
5. Answer any questions
6. Confirm handoff completion

**Step 3: Confirmation (Properties)**
1. Review handoff information
2. Verify completeness
3. Confirm understanding of scope
4. Identify any missing information
5. Acknowledge handoff receipt
6. Begin rehab planning

**Step 4: Follow-Up**
1. Properties team reviews property
2. Identifies any additional issues
3. Requests clarification if needed
4. Confirms rehab plan
5. Updates Acquisitions on timeline

### Handoff Best Practices

**Completeness:**
- Include all required information
- Don't assume Properties knows details
- Provide context and background
- Include all relevant documentation

**Accuracy:**
- Verify all information
- Ensure data is current
- Update any changes since underwriting
- Confirm property condition matches expectations

**Communication:**
- Clear and concise information
- Answer questions promptly
- Provide context for decisions
- Maintain open communication

**Timeliness:**
- Complete handoff within 24 hours
- Don't delay rehab start
- Communicate any delays
- Prioritize urgent properties

## Role Responsibilities

**Acquisitions Team**
- Prepare complete handoff package
- Complete handoff checklist
- Transfer property and information
- Answer Properties team questions
- Follow up as needed

**Properties Team**
- Review handoff information
- Verify completeness
- Confirm understanding
- Begin rehab planning
- Update Acquisitions on progress

**Operations**
- Monitor handoff completion
- Ensure SLA compliance
- Resolve handoff issues
- Optimize handoff process

## KPIs or SLAs

- **Handoff Completion**: 100% within 24 hours of closing
- **Handoff Completeness**: 95%+ include all required information
- **Handoff Accuracy**: 98%+ accurate information
- **Rehab Start Time**: Properties begins work within 48 hours of handoff

## Example Scenarios

**Scenario 1: Standard Handoff**
Property closes Monday. Acquisitions completes handoff checklist, gathers all documentation, transfers to Properties Tuesday morning. Properties reviews, confirms scope, begins rehab planning. Rehab starts Wednesday. Smooth handoff.

**Scenario 2: Complex Property**
Property has extensive repair needs, complex scope. Acquisitions prepares detailed handoff, schedules meeting with Properties. Review property together, discuss scope and priorities. Properties confirms understanding, begins detailed planning. Handoff completed successfully.

**Scenario 3: Missing Information**
Properties receives handoff, identifies missing inspection report. Contacts Acquisitions, requests report. Acquisitions provides report promptly. Properties proceeds with planning. Issue resolved quickly.
